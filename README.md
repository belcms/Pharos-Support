# Pharos - Support Page 📚✨

**Apple Swift Student Challenge 2026 Winner**

This repository is dedicated to hosting the static **Support** and **Privacy Policy** pages for the iOS application **Pharos: Immersive Reading**, intended for App Store submission.

## Live Site
You can access the active support page here:
[https://belcms.github.io/Pharos-Support/] 

## About Pharos
Pharos is your sanctuary for deep reading. The app combines immersive 3D environments, soothing soundscapes, and smart local AI tools (Foundation Models) to help users eliminate distractions, build focus, and completely immerse themselves in their reading sessions. 

## Repository Structure
* `index.html`: Main support page containing contact information.
* `icon2.png`: App icon visual asset used on the page.
* `privacy.html`: Dedicated page for the Privacy Policy terms.
